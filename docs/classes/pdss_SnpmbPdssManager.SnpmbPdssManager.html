Class SnpmbPdssManager

Short explanation of PDSS Service

I don't know if there is a clone or copy between the SNBP and PDSS services. The authentication flow of the PDSS service is the same as the SNBP service; there are no differences in authentication. Let's say if you want to copy the SNBP service flows to the PDSS service, you should change "snmptn" to "pdss" in every code or endpoint name.
